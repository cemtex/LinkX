Language File

Introduction

The LinkX language file allows you to translate error and other informative messages into your native language. This document describes how to modify the language file settings.

Control Panel Editing Interface

All changes to your language settings will be made through the control panel. To edit your language settings access the Templates > Language File function from the control panel menu. A page will load where you can edit all of the available language settings. Each of the items should be self-explanitory by reading the existing text. You can modify the settings to change the messages or translate them into your language. Once you have made the changes you want, press the Save Language File button at the top of the page.

Value Placeholders

Some of the default messages have value placeholders (%d, %s, etc) embedded in them so that dynamic information can be inserted into them. Each of the language file settings that has these placeholder values are described below.
    RATING_RANGE %d will be replaced by the maximum rating value you specified in the General Settings interface
    BLACKLIST_MATCHED The first %s will be the blacklist item that was matched
    The second %s will be the reason why this item is in the blacklist
    TITLE_LENGTH The first %d will be the minimum title length
    The second %d will be the maximum title length
    COMMENT_LENGTH The first %d will be the minimum comment length
    The second %d will be the maximum comment length
    DESCRIPTION_LENGTH The first %d will be the minimum description length
    The second %d will be the maximum description length
    MAXIMUM_KEYWORDS %d will be replaced by the maximum keywords value you specified in the General Settings interface
    BROKEN_URL The first %s is the type of URL submitted (either site URL or recip URL)
    The second %s is the error message associated with this URL
    COMMENT_LIMIT %d is the number of seconds a user must wait between comment submissions
