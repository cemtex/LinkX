Comments

Introduction

The LinkX comment system allows your users to submit messages that will be displayed on the details page for the link they are commenting on. You can restrict comments to only registered users and also use the blacklist feature to prevent unwanted items from appearing in your comments. This document will give an overview of managing user comments.

Search Comments

To access the comment search interface use the Directory > Search > Comments function from the control panel menu. This will load a page which will initially indicate that no search term has been entered. To start your search, enter the term you would like to search for in the text input box and select the field that you would like to search in. Each of the search fields is described below.

The Comment Text field supports boolean mode searching; the other fields support standard searching.
    Comment Text This will search in the text of the comment
    Username The username of the account the comment was submitted with, if any
    E-mail Address The e-mail address associcated with the comment
    Name The name associated with the comment
    Submitter IP The IP address used to submit the comment
    Date Added The date and time the comment was added (YYYY-MM-DD HH:MM:SS format)
    Comment ID The unique ID number assigned to the comment
    Link ID The current link status (one of unconfirmed, pending, active, or disabled)
In addition to searching the fields above, you can also select to display comments that have a specific status. Each of the possible status options are described below.
    All This will display all links, no matter what their status
    Pending This will display comments that have been submitted but are awaiting approval by an administrator
    Approved This will display links that have been approved and are currently displayed in the link directory
You can also choose how the comments are sorted by selecting the sorting field from the drop down list. The available sorting fields correspond with the search fields which are described above. Once you have the search values filled in how you want them, press the search button to display the comments that meet your criteria.

Comment Functions

In the search result you will be able to run a number of different functions on the comments that are displayed. To display additional information about the comment, click on the icon. This will send a request to your server for the information, so it may take a couple of seconds to load. Once it has been loaded, the information will slide into view.

Clicking on the icons in the Functions column of the table will allow you to run that function on a single comment. You can also check the boxes next to multiple comments to perform the functions listed at the bottom of the page. The single comment function icons are each described below.
    Click this icon to edit the comment. A window will appear where you will be able to edit all of the comment settings.
    Click this icon to approve the comment (visible only on pending comments)
    Click this icon to reject the comment (visible only on pending comments)
    Click this icon to view the user account that submitted the comment (visible only on comments submitted by an account)
    Click this icon to view the link that this comment is associated with
    Click this icon to delete the comment. You will be asked to confirm the deletion before it is done.
You can also run a function on multiple comments by putting a check in the box next to the comments you want, selecting the function to run from the drop down selection field at the bottom of the page, and pressing the Execute button. Each of the available functions are described below.
    Delete Selected This will permanently delete the comments you have selected
    Approve Selected This will approve the comments you have selected. (this function should be used only for pending comments)
    Reject Selected This will reject the comments you have selected. (this function should be used only for pending comments)
Processing New Comments

If you have the option enabled to require approval of new comments, you will need to login to the control panel from time-to-time to review newly submitted comments. To view the newly submitted comments, select To Do > Review Comments from the control panel menu. This will load the Comment Search interface with the appropriate options set to display only comments with pending status.

New comments can be processed one at a time, or in bulk. To process them one at a time click on the icon to approve the comment or the icon to reject the comment.

To process many comments at once, put a check in the box next to the comments you want to process. At the bottom of the page select either 'Approve Selected' or 'Reject Selected' from the drop down list of functions. Press the Execute function to process all of the selected comments using the function you have choosen.

Editing a Comment

To edit a comment you will fist need to locate it through the Comment Search interface. When you locate the link you want to edit, click on the icon to bring up the link editing interface. This will display a popup window where you will be able to update the comment data. Each of the form fields are described below.
    Name The name of the comment submitter
    E-mail The e-mail address that is associated with this comment
    Date Added The date this comment was added (must be in YYYY-MM-DD HH:MM:SS format)
    Comment The comment itself
Once you have modified all of the fields as you want them, press the Update Comment button.

Deleting a Comment

To delete a comment you will fist need to locate it through the Comment Search interface. When you locate the comment you want to delete, click on the icon, which will ask you to confirm the deletion.

You can also delete several comments at once by putting a check in the box next to the comments you want to delete, selecting the 'Delete Selected' function from the drop down list at the bottom of the page and pressing the Execute button.
