General Settings

Introduction

The General Settings interface allows you to configure the software options as you like. There are a number of settings that you can adjust, and each of them is described in this section of the manual.

Basic Settings
    Base URL - This is the URL that points to the directory where LinkX is installed. If you install LinkX in the base directory of your site this would be http://www.yoursite.com Cookie Domain - This is the domain name of the site where LinkX is installed, for use in setting cookies

    E-mail Address - This is the e-mail address that will appear in the From field of all e-mails sent by LinkX

    E-mail Name - This is the name that will appear in the From field of all e-mails sent by LinkX

    E-mail Sender - Choose the method that LinkX will use for sending emails. In most cases you should leave this set to the default PHP mail() Function option.

      PHP mail() Function - LinkX will use PHP's built in mail() function for sending e-mail messages.
      Sendmail - LinkX will use sendmail on your server to send e-mail messages. This will require that PHP is not running in safe mode and the popen() function is not disabled. You will need to enter the full path to sendmail in the text input box after the drop down selection field.
      SMTP Server - LinkX will use a SMTP server to send e-mail messages. This will require that the SMTP server has authorized your server to send e-mail messages through it. You will need to enter the hostname of the SMTP server in the input box after the drop down selection field.


    New Links Page - The name you want to use for the page that will display the newest links. If you are using mod_rewrite to create search engine friendly URLs, this page name will appear directly in the URL. If you are not using mod_rewrite, this page name will appear in the index.php query string.

    Popular Links Page - The name you want to use for the page that will display the most clicked links. If you are using mod_rewrite to create search engine friendly URLs, this page name will appear directly in the URL. If you are not using mod_rewrite, this page name will appear in the index.php query string.

    Top Links Page - The name you want to use for the page that will display the top rated links. If you are using mod_rewrite to create search engine friendly URLs, this page name will appear directly in the URL. If you are not using mod_rewrite, this page name will appear in the index.php query string.

    Link Details Page - The name you want to use for the page that will display the link details. You must place a %d somewhere in this page name, which will be replaced by the link ID number dynamically by LinkX. This value is only used when you have mod_rewrite enabled for search engine friendly URLs

    File Extension - The file extension to use for link pages. This value is only used when you have mod_rewrite enabled for search engine friendly URLs

    Date Format - The string that specifies the format to use for date values

    Time Format - The string that specifies the format to use for date values

    Timezone - Select the timezone you are located in so that dates and times can be calculated properly

    Decimal Point - The value to use for the decimal point in numbers

    Thousands Separator - The value to use for the thousands separator in numbers

    Use mod_rewrite to create search engine friendly URLs - If your server has the mod_rewrite Apache module enabled, you can check this box if you want search engine friendly URLs to be used.

    Using custom templates for category pages - Check this box if you have configured a custom template name for any of your categories. Using custom category templates will require additional database queries, so this is only recommended if you absolutely need it and cannot create a custom template using the already available template commands.
Link Submission Settings
    Description Length - The minimum and maximum number of characters allowed in a link description

    Title Length - The minimum and maximum number of characters allowed in a link title

    Keywords Allowed - The maximum number of keywods that can be submitted with a link

    Default Link Weight - The default link weight value assigned to newly added links

    Allow URL redirection - Check this to allow links to be submitted that redirect to other URLs

    Require a reciprocal link - Check this to require that submitted links contain a reciprocal link to your site

    Only registered users can submit links - Check this to only allow registered users to submit links

    Send confirmation e-mail message to new lnks - Check this to send a confirmation e-mail to the submitter of a new link

    New links must be confirmed through e-mail before activation - Check this to require that new link submissions be confirmed through the e-mail address of the submitter

    Edited links must be approved by an administrator before activation - Check this to require that all user link modifications be approved by an administrator before appearing in the link directory
Comment Settings
    Comment Length - The minimum and maximum number of characters allowed in a comment

    Comment Delay - The amount of time, in seconds, that a user must wait before submitting another comment for the same link

    Only registered users can leave comments for links - Check this to only allow registered users to submit comments

    New comments must be approved by an administrator before display - Check this to require that new comments be approved by an administrator before they appear
Rating Settings
    Maximum Rating - The maximum (best) rating that a link can receive. 5 is the recommended value because LinkX ships with icons that are setup for a rating range from 1 to 5 stars.

    Only registered users can rate links - Check this to only allow registered users to submit ratings

Account Settings
    Send confirmation e-mail message to new accounts - Check this to have LinkX send a confirmation e-mail message to all newly created accounts

    New accounts must be confirmed through e-mail before activation - Check this to require that new link accounts be confirmed through the e-mail address of the submitter

    New accounts must be approved by an administrator before activation - Check this to require that all new user accounts be approved by an administrator before becoming active
dsbl.org Blacklist Settings

dsbl.org (Distributed Sender Blackhole List) provides a service where you can find if certain IP addresses have been blocked because they are known to be on open proxy, e-mail relay, or other compromised computer. Many times these IP addresses represent computers that have been hacked and are being controlled by spammers to allow them to make HTTP requests from many different IP addresses. Enabling any of these options will cause LinkX to check against the dsbl.org list when the specified action is taking place, and will block the user from completing that action if their IP address is on the list.
    Check dsbl.org for banned IP addresses during link submission
    Check dsbl.org for banned IP addresses during link rating
    Check dsbl.org for banned IP addresses during link comment submission
    Check dsbl.org for banned IP addresses during account creation
    Check dsbl.org for banned IP addresses during bad link reporting
Verification Code Settings

Please also review the Verification Codes section of this manual
    Font Directory - The full directory path to the location of the LinkX fonts directory.

    Code Length - The minimum and maximum number of characters that will appear in a verification code.

    Use words file for verification codes - Check this box to use the values stored in the words file for the verification codes.

    Require verification code on link submission form - Check this box to require a verification code during link submission.

    Require verification code on account signup form - Check this box to require a verification code during account signup.

    Require verification code to rate links - Check this box to require a verification code during link rating.

    Require verification code to leave comments - Check this box to require a verification code during comment submission.
Cache Settings

Please also review the Cache System section of this manual
    Index Page - The time, in seconds, between cache updates of the main link directory page.

    Category Page - The time, in seconds, between cache updates of the link category pages.

    New Links Page - The time, in seconds, between cache updates of the new links page.

    Popular Links Page - The time, in seconds, between cache updates of the most clicked links page.

    Top Links Page - The time, in seconds, between cache updates of the top rated links page.

    Search Page - The time, in seconds, between cache updates of the link search page.

    Details Page - The time, in seconds, between cache updates of the link details page.
