LinkX Documentation
General Information

This documentation covers the LinkX software functions, and tries to be as thorough as possible. If there is something you cannot find in this manual, please check the knowledge base first. If you still cannot find an answer to your question, you may contact the tech support staff through the owners lounge. We have tried to make the information in this manual as accurate as possible, however if you do find any errors or inconsistencies, please let us know so we can correct it.

If you have just had LinkX installed for the first time, please see the First Steps section of this manual first. It will help get you started using LinkX and will guide you through the first things you need to do.
